# Event Definitions

Per-flow event definitions for Helix analytics. Each flow folder contains the complete specification for implementing and analyzing events.

## Structure

Each flow folder contains:

| File | What it covers |
|------|---------------|
| `events.md` | Event catalog — names, types, sources, triggers |
| `properties.md` | Property dictionary — types, allowed values, which events use them |
| `firing-rules.md` | When and why each event fires — behavioral proof, drop-off scenarios |
| `analytics-guide.md` | Funnels, segmentation, questions each event answers |
